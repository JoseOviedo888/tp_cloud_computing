# To-Do App (Gestor de Tareas Personal)

## Descripción
Una aplicación simple en Python con interfaz gráfica para crear, completar y eliminar tareas personales. Los datos se almacenan en un archivo JSON.

## Integrantes
- Tu Nombre – tucorreo@uninorte.edu.py

## Funcionalidades
- Agregar tareas
- Marcar tareas como completadas
- Eliminar tareas
- Guardado en JSON automático

## Cómo ejecutar
```bash
git clone https://github.com/JoseOviedo888/todo-app.git
cd todo-app
python main.py
